Welcome to the Tosca Automation Test Project

This project demonstrates automated testing using Tricentis Tosca.

Getting Started

  1. Clone the subset to your local machine.
  2. Install and configure Tricentis Tosca on your system.
  3. Open Tosca and import the subset to the project directory.
  4. Create and configure Tosca test cases.
  5. Run automated tests using Tosca.

For more detailed information, refer to the project's README file.
